# Similarity Viewer

Dedicated static frontend for similarity analysis reports generated by `--similarity-report`.

## Run

Serve the repository via HTTP and open `/tools/similarity_viewer/`.

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000/tools/similarity_viewer/>.

## Data Sources

- **Load Default**: tries `../../build/similarity_report.json`.
- **Upload JSON**: load any report file matching the schema.

## Report Contract

Top-level fields consumed:
- `generatedAt`
- `modelId`
- `threshold`
- `topK`
- `groupCount`
- `groups[]`

Each group must contain:
- `anchorPath`
- `neighborCount`
- `neighbors[]` with `path` and `score`

## Features

- Search by anchor or neighbor path.
- Minimum score filter.
- Group list sorted by neighbor count.
- Pair table sorted by score descending and then path ascending.
- Copy filtered pairs as JSON.
