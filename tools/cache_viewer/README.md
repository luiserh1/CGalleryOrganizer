# Cache Viewer Dashboard

Static dashboard for inspecting cache JSON files generated by CGalleryOrganizer.

## Setup

Because the app fetches JSON files, serve the project over HTTP (do not open `index.html` directly).

### Option A (Python)
```bash
python3 -m http.server 8000
```
Then open: <http://localhost:8000/tools/cache_viewer/>

### Option B (Node)
```bash
npx serve .
```
Then navigate to `/tools/cache_viewer/`.

## Default file behavior

The **Load Default** button tries to load:
- `build/perf_cache.json`

Generate it via stress run:
```bash
make stress
```

## Sorting behavior (v0.2.4)

Column sorting uses real backing fields:
- Path -> `path`
- Size -> `fileSize`
- MD5 Hash -> `exactHashMd5`
- Dimensions -> width/height area
- Date Taken -> EXIF date fallback to `modificationDate`
- Camera -> `cameraMake + cameraModel`

## Cleanup

```bash
make clean      # removes binaries and generated build outputs
make clean-all  # removes full build directory, including downloaded stress data
```

> Note: the dashboard does not mutate production cache files unless you explicitly replace/upload data.
