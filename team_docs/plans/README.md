# Cycle Plans Directory

This directory stores PM-authored implementation plans for each active cycle.

## Naming Convention
- `YYYY-MM-DD_<cycle-name>_IMPLEMENTATION_PLAN.md`

## Required Plan Content
- Objective and scope boundaries
- Role assignment by task slice
- Task order and dependencies
- Acceptance owner for each slice
- Merge gates and validation checks
- Open risks and escalation path

## Governance
- PM authors plan content.
- Main Maintainer is the final merger/committer for PM-authored `team_docs/` documents.
